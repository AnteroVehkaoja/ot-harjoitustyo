# Käyttöohje

lataa ohjelmiston viimeisin versio

## konfigurointi

muuta .config tidostossa olevia muuttujia height, width ja GEN_GRID, mikä luo ruudukon jossa ei tarvitse arvata

## käynnistys

mene projektin juurihakemistoon 



```bash
poetry install
```
ja käynnistä komennolla 

```
poetry run invoke start
```

nyt voit pelata miinaharavaa, tulokset tallettuu results hakemiston tiedostoon
